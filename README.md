# Telegram-бот: Сборщик списков покупок 🛒

Этот Telegram-бот помогает быстро и удобно спланировать покупки на неделю. Вы выбираете блюда из предложенного меню, а бот автоматически подсчитывает необходимое количество ингредиентов и формирует итоговый список покупок.

## Функционал

- Интерактивное меню с выбором блюд (с использованием Inline-клавиатуры).
- Автоматический подсчет веса (в граммах) и количества (в штуках) продуктов.
- Объединение одинаковых ингредиентов из разных блюд в общую сумму.
- Удобный вывод готового списка покупок.

## Стек технологий

- Python 3.11+
- [aiogram 3.x](https://docs.aiogram.dev/en/latest/) (Асинхронный фреймворк для Telegram Bot API)
- [python-dotenv](https://pypi.org/project/python-dotenv/) (Для работы с переменными окружения)

## Установка и запуск

1. **Клонируйте репозиторий (или скачайте проект):**
   ```bash
   git clone <URL_репозитория>
   cd <Имя_папки>
   ```

2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   # Windows
   python -m venv venv
   venv\Scripts\activate

   # Linux/macOS
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройте переменные окружения:**
   - Переименуйте (или скопируйте) файл `.env.example` в `.env`.
   - Откройте файл `.env` и вставьте токен вашего бота (его можно получить у [@BotFather](https://t.me/BotFather)):
     ```env
     BOT_TOKEN=ваш_токен_бота
     ```

5. **Запустите бота:**
   ```bash
   python bot.py
   ```

## Структура проекта

- `bot.py` — главный файл с логикой бота.
- `recipes.json` — база данных рецептов с граммовками (вы можете легко дополнять её своими рецептами).
- `requirements.txt` — список зависимостей.
- `.env.example` — пример файла с переменными окружения.

## Добавление и настройка рецептов

Вы можете легко добавлять свои собственные блюда, редактируя файл `recipes.json`. 

Пример формата:
```json
{
  "Ваше новое блюдо": {
    "Ингредиент 1": 500,
    "Ингредиент 2": 2
  }
}
```

**Особенности подсчета:**
- По умолчанию все единицы измерения считаются **в граммах**.
- Ингредиенты `Яйцо` и `Лук` считаются **в штуках**. При необходимости это поведение можно изменить, отредактировав множество `PIECE_UNITS` внутри файла `bot.py`.
